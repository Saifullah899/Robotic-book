## About

This textbook provides an introduction to Physical AI and Humanoid Robotics, covering fundamental concepts from basic robotics principles to advanced topics in humanoid locomotion and manipulation. The content is designed for beginners and intermediate learners interested in understanding the intersection of artificial intelligence and robotics.

## Features

- 6 comprehensive chapters covering essential topics in Physical AI & Humanoid Robotics
- Accessible to beginners while providing depth for intermediate learners
- Responsive design for mobile and desktop viewing
- Offline capability after initial visit
- Clean, accessible interface with proper navigation

## Development

### Installation

```bash
cd website
npm install
```

### Local Development

```bash
cd website
npm start
```

This command starts a local development server and opens the website in your browser. Most changes are reflected live without restarting the server.

### Build

```bash
cd website
npm run build
```

This command generates static content into the `build` directory and can be served using any static hosting service.

### Deployment

The site is configured for GitHub Pages deployment:

```bash
cd website
GIT_USER=<Your GitHub username> npm run deploy
```
